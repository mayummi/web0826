1. Node.js 설치
-> npx(Node Package eXecute), npm(Node Package Manager) 명령어 입력
-> 출처: https://youngmin.hashnode.dev/npm-npx

2. Node 설치 확인
명령줄 인터페이스(CLI)는 Command-Line Interface
    가. ctrl+` -> 터미널
    나. 에러가 날 경우는 컴퓨터 재부팅후 확인

실습>
web0826>node -v
결과: v20.17.0
web0826>npx -v
결과: 10.8.2
web0826>npm -v
결과: 10.8.2
